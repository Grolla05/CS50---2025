Description and idea of my project

This project was developed to be a WEB application, using the following technologies: Python for the back-end, HTML, TailWind CSS and JavaScript for the front-end of the WEB pages.
The main objective of this web application is to be an automatic product catalog in which, when inserted into the database, through software also developed in Python, it is possible for it to be displayed on the page, as can be seen.
The usefulness of this application is aimed at e-commerces such as Shopee, Mercado Livre, AliExpress, Amazon...
